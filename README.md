# domotica

An ecleptic varity of project used at home;



**Solar**

En vez de comprar el kit de [Autosolar](https://autosolar.es/kit-solar-aislada/kit-solar-casa-campo-3000w-24v-6400whdia), ponemos un inversor chino por 1700 euros.

| Concepto | Precio |
| -------- | ------ |
| Inversor | 255    |
| Paneles  | 2x70   |
| Soporte  | 64     |
| Baterias | 2x200  |

En el garaje, el motor se arranca cuando queramos inyectar agua al deposito y/o cargar las baterias.

- *Problema*: no se sabe por donde esta el nivel!
- **Reuso**: Panel 12V con MPTT para Router 4G.
